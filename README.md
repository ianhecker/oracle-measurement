# Oracle Measurement

## Setup

### NPM
Run
```
npm install
```

### Secrets
Create a secrets file with a password
```
npx env-enc set-pw
```

Now create the following secrets with `npx env-enc set`
```
ALCHEMY_API_KEY
```
```
COINMARKETCAP_API_KEY
```
```
ETHEREUM_SEPOLIA_RPC_URL
```
```
PRIVATE_KEY
```
```
TOMORROW_IO_API_KEY
```
