# Oracle Measurement

## Setup

### NPM
Run

```
npm install
```

### Secrets

Create a secrets env file. Fill in the values

```bash
cat << EOF > .env
ALCHEMY_API_KEY = ""
COINMARKETCAP_API_KEY = ""
ETHEREUM_SEPOLIA_RPC_URL = ""
PRIVATE_KEY = 0x
TOMORROW_IO_API_KEY = ""
EOF
```

npm install dotenv
